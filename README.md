## Sheep QRCODE GETTER

powered by jQuery & jquery-qrcode.

---

## Icons
Drawed a fat sheep and add a qrcode by myself.


## Install

```
clone or download this repo -> open chrome -> open the url: chrome://extensions -> Load unpacked extension... -> Select the directory
```
